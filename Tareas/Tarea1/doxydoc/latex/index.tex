Bienvenido a la documentación de los programas en C++ desarrollados como parte del proyecto. A continuación se presenta un resumen de los dos programas principales incluidos en este proyecto.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Programa de Conversión de Unidades}{Programa de Conversión de Unidades}}\label{index_autotoc_md1}
Este programa permite a los usuarios realizar conversiones entre diferentes unidades de medida en tres categorías principales\+: longitud, peso y temperatura. Utiliza un menú interactivo que permite al usuario seleccionar el tipo de conversión, ingresar el valor a convertir y obtener el resultado en la unidad deseada.

{\bfseries{Opciones del menú principal\+:}}


\begin{DoxyItemize}
\item Convertir entre unidades de longitud (metros, kilómetros, etc.).
\item Convertir entre unidades de peso (gramos, kilogramos, etc.).
\item Convertir entre unidades de temperatura (Celsius, Fahrenheit, Kelvin).
\item Salir del programa.
\end{DoxyItemize}

El programa utiliza Enums, Structs, sobrecarga de funciones y recursividad para realizar las conversiones.\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Calculadora de Estadísticas y Búsqueda en un Arreglo}{Calculadora de Estadísticas y Búsqueda en un Arreglo}}\label{index_autotoc_md2}
Este programa permite calcular estadísticas básicas (suma, promedio, máximo y mínimo) y realizar una búsqueda de un número específico dentro de un conjunto de números. Los números se reciben como argumentos desde la línea de comandos y se almacenan en un arreglo utilizando punteros.

{\bfseries{Opciones del menú principal\+:}}


\begin{DoxyItemize}
\item Mostrar estadísticas (suma, promedio, máximo y mínimo).
\item Buscar un número en el arreglo.
\item Salir del programa.
\end{DoxyItemize}

Las estadísticas se calculan utilizando funciones con punteros, y el programa permite realizar búsquedas de manera eficiente.

{\bfseries{Notas\+:}}


\begin{DoxyItemize}
\item Ambas aplicaciones están diseñadas para ser ejecutadas en un entorno de consola y proporcionan una interfaz interactiva al usuario. 
\end{DoxyItemize}
