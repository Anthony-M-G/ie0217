\doxysubsection{Detailed Description}
Representa una tienda que gestiona productos y clientes. 

La clase {\ttfamily \doxylink{classTienda}{Tienda}} administra una lista de productos y clientes. Permite agregar productos y clientes, y procesar compras para clientes específicos.
