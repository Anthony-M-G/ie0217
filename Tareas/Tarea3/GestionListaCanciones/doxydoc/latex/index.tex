\chapter{2. Gestión de una Lista Enlazada Dinámica de Canciones}
\hypertarget{index}{}\label{index}\index{2. Gestión de una Lista Enlazada Dinámica de Canciones@{2. Gestión de una Lista Enlazada Dinámica de Canciones}}
\label{index_md_index}%
\Hypertarget{index_md_index}%


Este programa implementa una lista enlazada dinámica para gestionar una colección de canciones. Cada nodo de la lista almacena un objeto de la clase {\ttfamily \doxylink{classCancion}{Cancion}}, que contiene información básica sobre la canción (nombre, artista, duración). El programa permite\+:


\begin{DoxyItemize}
\item Inicializar una lista con 3 canciones predefinidas.
\item Insertar nuevas canciones al inicio, final o en posiciones intermedias.
\item Eliminar canciones de la lista.
\item Modificar los datos de una canción existente.
\item Imprimir la lista completa de canciones.
\end{DoxyItemize}\hypertarget{index_autotoc_md1}{}\doxysubsubsection{\texorpdfstring{Operaciones Disponibles\+:}{Operaciones Disponibles\+:}}\label{index_autotoc_md1}

\begin{DoxyEnumerate}
\item Insertar una nueva canción.
\item Eliminar una canción.
\item Modificar los datos de una canción existente.
\item Imprimir la lista de canciones.
\item Salir del programa (liberando toda la memoria dinámica). 
\end{DoxyEnumerate}
