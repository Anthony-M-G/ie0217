\doxysubsection{Detailed Description}
Clase que representa una lista enlazada de canciones. 

La clase \doxylink{classListaCanciones}{Lista\+Canciones} gestiona una colección de canciones usando una lista enlazada. Proporciona métodos para insertar, eliminar, modificar y listar canciones.
