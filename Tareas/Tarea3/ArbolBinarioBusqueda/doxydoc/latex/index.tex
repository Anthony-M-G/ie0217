Este programa implementa un Árbol Binario de Búsqueda (ABB) en C++. El árbol inicial se crea automáticamente con una estructura predefinida de nodos. El programa permite\+:


\begin{DoxyItemize}
\item Inserción y eliminación de nodos.
\item Verificación de que el árbol sigue siendo un ABB válido.
\item Cálculo de la altura del árbol.
\item Determinación de si el árbol está balanceado.
\item Recorrido e impresión del árbol en orden ascendente.
\end{DoxyItemize}\hypertarget{index_autotoc_md1}{}\doxysubsubsection{\texorpdfstring{Operaciones Disponibles\+:}{Operaciones Disponibles\+:}}\label{index_autotoc_md1}

\begin{DoxyEnumerate}
\item Insertar un nodo.
\item Eliminar un nodo.
\item Imprimir el árbol.
\item Calcular la altura del árbol.
\item Verificar si el árbol está balanceado.
\item Salir del programa. 
\end{DoxyEnumerate}
