Detailed Description

Clase que proporciona servicios de gestión de reseñas en la base de datos.

Esta clase implementa la capa de servicio para manejar todas las operaciones relacionadas con reseñas, incluyendo la gestión de estados, consultas de calificaciones y eliminación de reseñas.
