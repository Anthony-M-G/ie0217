Detailed Description

Clase para manejar la conexión a una base de datos MySQL.

Esta clase proporciona métodos para establecer y obtener una conexión a una base de datos MySQL. Utiliza el controlador MySQL de la biblioteca mysqlcppconn.

Note
Asegúrese de tener la biblioteca mysqlcppconn correctamente instalada y configurada.
