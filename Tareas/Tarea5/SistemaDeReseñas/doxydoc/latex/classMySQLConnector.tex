\doxysubsection{Detailed Description}
Clase para manejar la conexión a una base de datos My\+SQL. 

Esta clase proporciona métodos para establecer y obtener una conexión a una base de datos My\+SQL. Utiliza el controlador My\+SQL de la biblioteca {\ttfamily mysqlcppconn}.

\begin{DoxyNote}{Note}
Asegúrese de tener la biblioteca {\ttfamily mysqlcppconn} correctamente instalada y configurada. 
\end{DoxyNote}
