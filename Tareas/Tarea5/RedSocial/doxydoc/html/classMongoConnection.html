Detailed Description

Esta clase maneja la conexión a una base de datos MongoDB y proporciona métodos para interactuar con ella.

La clase MongoConnection se encarga de inicializar la instancia de MongoDB, crear el cliente para conectarse a la base de datos y proporcionar acceso a las colecciones de la base de datos.
