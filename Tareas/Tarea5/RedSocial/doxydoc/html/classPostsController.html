Detailed Description

Controlador para manejar las solicitudes relacionadas con las publicaciones.

La clase PostsController es responsable de recibir y procesar solicitudes para gestionar las publicaciones en la red social. Llama a los métodos de PostsServices para interactuar con la base de datos.
