\doxysubsection{Detailed Description}
Controlador para manejar las solicitudes relacionadas con las publicaciones. 

La clase {\ttfamily \doxylink{classPostsController}{Posts\+Controller}} es responsable de recibir y procesar solicitudes para gestionar las publicaciones en la red social. Llama a los métodos de {\ttfamily \doxylink{classPostsServices}{Posts\+Services}} para interactuar con la base de datos.
