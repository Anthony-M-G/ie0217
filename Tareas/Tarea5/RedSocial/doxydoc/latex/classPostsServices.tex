\doxysubsection{Detailed Description}
Clase que maneja las operaciones relacionadas con las publicaciones en una red social. 

La clase {\ttfamily \doxylink{classPostsServices}{Posts\+Services}} proporciona métodos para gestionar publicaciones en una base de datos Mongo\+DB, permitiendo insertar, consultar, actualizar, eliminar publicaciones, y manejar comentarios.
