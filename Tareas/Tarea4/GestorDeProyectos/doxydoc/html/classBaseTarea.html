Detailed Description

Clase base para tareas genéricas con diferentes tipos de recursos en un proyecto.

Esta clase define los atributos y métodos básicos necesarios para representar una tarea. Está diseñada para ser heredada por la clase Tarea, permitiendo la creación de tareas con diferentes tipos de recursos.
