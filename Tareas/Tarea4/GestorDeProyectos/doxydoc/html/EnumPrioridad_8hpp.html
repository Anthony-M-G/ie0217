Detailed Description

Define la enumeración Prioridad para la priorización de tareas.

Este archivo de encabezado contiene la definición de la enumeración Prioridad, que se utiliza para especificar el nivel de prioridad de las tareas en el gestor de proyectos. Los niveles de prioridad se definen de la siguiente manera:

  • ALTA: Alta prioridad (valor 1)
  • MEDIA: Prioridad media (valor 2)
  • BAJA: Baja prioridad (valor 3)
